Zgcam(Gcam mod for Exynos Processor-based Devices)

As Gcam(Google Camera) is for Pixel devices only, Snapdragon-based. The code for Gcam was only compatible and optimized to work with the Snapdragon processors; Moreover, the Snapdragon processors have dedicated image processing units for Gcam called Hexagon DSP, which was lacking in their Exynos counterparts. We are a group of developers who helped Zoran Ostojic mod Gcam and made it work on Exynos processors overcoming the challenges like – Lack of Hardware (Hexagon DSP), computational power, and code only optimized to work on Snapdragon processor.
